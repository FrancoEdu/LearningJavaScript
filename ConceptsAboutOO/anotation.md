# Conceitos de Orientação à Objetos
===================================

## Métodos
==========
* Proppriedades que servem como funções
* Ou seja, as ações dos objetos
* Invocamos os métodos da mesma maneira que funções

<code>
    let chachorro = {
        latir: function(){
            console.log("Au au");
        }
    }
    cachorro.latir();
</code>
